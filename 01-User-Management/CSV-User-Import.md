# CSV User Import

## Overview

CSV User Import is a feature in Identity and Access Management (IAM) 
platforms that allows administrators to create multiple user accounts 
simultaneously using a Comma-Separated Values (CSV) file.
This simplifies bulk onboarding and reduces manual effort.

---

## Why Use CSV Import?

- Create multiple users at once
- Save time during employee onboarding
- Reduce manual data entry
- Minimize human errors
- Standardize user information

---

## Common CSV Fields

| Field | Description |
|--------|-------------|
| First Name | User's first name |
| Last Name | User's last name |
| Username | Unique login name |
| Primary Email | User's email address |
| Department | Business department |
| Employee ID | Unique employee identifier |
| Job Title | User's designation |
| Manager | Reporting manager |

---

## CSV Import Process

1. Prepare a CSV file with the required user information.
2. Navigate to the user import section in the IAM platform.
3. Upload the CSV file.
4. Validate the data.
5. Resolve any validation errors.
6. Import the users.
7. Verify that all user accounts have been created successfully.
