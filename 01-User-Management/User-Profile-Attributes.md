# User Profile Attributes

## Overview

User profile attributes store information about a user's identity within an Identity and Access Management (IAM) system. These attributes help identify users and support authentication, authorization, and provisioning.

---

## Common User Attributes

- First Name
- Last Name
- Username
- Primary Email
- Employee ID
- Department
- Job Title
- Manager
- Phone Number
- Country
- City
- User Status

---

## Why Are User Attributes Important?

User attributes are used to:

- Identify users uniquely
- Authenticate users
- Authorize access to applications
- Synchronize identities with external directories
- Support user provisioning and lifecycle management
