# Creating Users

## Objective

Create a new user in the IAM platform.
## Steps

1. Navigate to **Directory → People**.
2. Click **Add Person**.
3. Enter the required user details:
   - First Name
   - Last Name
   - Username
   - Primary Email
4. Save the user.

## Result

A new user account is successfully created and is available for identity management
tasks.
## Interview Questions

### What is user provisioning?
User provisioning is the process of creating, 
updating, and managing user accounts and granting appropriate access to enterprise resources.
### Why is user creation important in IAM?
User creation establishes a unique digital identity for an individual, 
enabling secure authentication, authorization, 
and access management within an organization.
## Why is User Creation Important?
User creation is the first step in Identity and Access Management (IAM).
Every employee requires a unique digital identity to authenticate,
access enterprise applications, and perform their job responsibilities securely.
