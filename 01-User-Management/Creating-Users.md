# Creating Users

## Objective

Create a new user in the IAM platform.

---

## Steps

1. Navigate to **Directory → People**.
2. Click **Add Person**.
3. Enter the required user details:
   - First Name
   - Last Name
   - Username
   - Primary Email
4. Save the user.

---

## Result

A new user account is successfully created and is available for identity management tasks.

---

## Why is User Creation Important?

User creation is the first step in Identity and Access Management (IAM). 
Every employee requires a unique digital identity to authenticate, access enterprise applications, and perform their job responsibilities securely.
