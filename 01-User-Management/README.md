# User Management

## Overview

User Management is the process of creating, managing, updating, and deleting user accounts within an Identity and Access Management (IAM) system.

---

## Topics Covered

- Creating Users
- User Profile Attributes
- User Status
- CSV User Import
- User Lifecycle Management
- User Administration

---

## Objectives

- Understand how users are created in an IAM platform.
- Learn common user profile attributes.
- Perform bulk user creation using CSV.
- Understand the user lifecycle.
- Gain hands-on experience with enterprise IAM user administration.

---

## Hands-on Lab

- Created users manually.
- Managed user profiles.
- Imported users using CSV.
- Verified user information.
- Assigned users to groups.

---

## Skills Gained

- User Administration
- Identity Management
